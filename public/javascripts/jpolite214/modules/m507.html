Remove Features You Don't Need

Not all features included in JPolite2 are needed in a tailored system, here's a simple check list to lose weight:

  • Navigation Tab Features - remove Kwicks and/or LavaLamp plugin files from index.html
  • jQuery UI & Theme - remove jquery-ui-1.7.2.custom.css and jquery-ui-1.7.2.custom.min.js if you don't need them. Note that module drag-n-drop requires jQuery UI js code. In that case build a custom download here: jQuery UI Download
  • Custom jQuery Plugins - remove plugin JS files and associated CSS files if any.
  • XDO - If you're not building a RESTful Ajax application, remove xdo.core.js, xdo.data.js, and chain-0.2.pack.js

Last not least, combine multiple JS and CSS files and minimize them to save space and load time!

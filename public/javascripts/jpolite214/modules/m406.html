Below is the example flow to integrate the jqDialog plugin

  1. First download jqModal.js and supporting CSS, and include them into index.html
  2. Initialization code in jpolite.ext.js 
    $.addControl({
	...
	".jqmWindow":	[$.fn.jqm, {toTop:true}],
	...
});
  3. Sample HTML segment within this module 
    <div class="jqmWindow" id="test1">
	<h4 class='success'>A sample jqModal is shown!</h4>
</div>
<div class="jqmWindow" id="test2">
	<h4 class='notice'>Another jqModal is here!</h4>
</div>

A Local Link is one that changes the content of its parent module only

<a href='modules/m502.html' class='local'>m502</a>
<a href='modules/m503.html' class='local'>m503</a>

Go to module Tabs Control
Go to module Accordion Control

Just add a class="local" attribute and it will work.

Web pages normally don't need this feature, but when loaded as a module under JPolite, links without the "local" mark will affect the entire page by default.

A local link only switches the content of its parent module into another's, thus a refresh will revert the original content.

External links, e.g., href=http://xyz.com/abc.html, will open a new window or tab by default in JPolite.

About jQuery User Interface Plugins

Currently there some 500+ UI plugins available for jQuery based systems. To integrate those you like into JPolite, you need to do the following:

  1. Download plugin JS file (and necessary supporting stuff) and include it in index.html, before jpolite.ext.js
  2. Call $.addControls({selector:[$.fn.myPluginFunction, {optional:init_arguments}]}), you can find examples in function myControls in jpolite.ext.js
  3. Include HTML sections pre-formatted according to plugin requirements into a module and test if it works

Step 2 seems a bit more complex, that's because modules in JPolite are loaded through the "lazy-loading" mechanism that one-time initialization (usually done in $(function(){...})) can notcover controls to be loaded in the future.

So JPolite2 introduces a control registry, which can be accessed via $.addControls. All registered controls' initialization methods will be automatically called upon each and every newly loaded module (or newly generated content).

Try to use class selector all the time, since a type of control may appear many times in many modules.

Beware that some plugins only support one-time initialization, that later calls on new content would not work.

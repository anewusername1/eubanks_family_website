Lastly, define class of the columns for the tabs in _columnLayout variable, e.g.,

...
_default: {
	bg:'normal',
	c1:'span-8',
	c2:'span-8',
	c3:'span-8 last',
	c4:'span-24 last'
},
t2:{	bg:'hehe',
	 c1:'span-24 last',
	 c4:'span-8 last'},
...
  • _default - default class definitions of columns
  • t2, t3 - class definitions for a given tab
  • bg - class definition for body, which will be switched when changing tab
  • c1, c2, c3 - column class definitions

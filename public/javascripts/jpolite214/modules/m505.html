BlueTrip CSS Base

JPolite 2 is based on BlueTrip CSS framework that 4 CSS files are used:

  • screen.css - BlueTrip's screen stylesheets.
  • print.css - BlueTrip's print stylesheets.
  • ie.css - BlueTrip's IE hacks.
  • styles.css - JPolite2's native stylesheets

It is recommended that you keep YOUR styles separate from BlueTrip's during development phase, to make it easier for management. And you'd combine and compress CSS files to save server bandwidth on production systems.
